# inventory-application
This is a project from The Odin Project curriculum to learn Express.js and Postgres.

## Run locally
To run this project locally, follow these steps:
1. Clone this repo and cd into it
2. Install from package.json (npm install)
3. Create a database in postgres
4. Create a .env file, using .env.example as reference
5. Run the project with node (node app.js)
